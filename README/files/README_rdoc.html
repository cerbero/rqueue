rqueue

rqueue is a simple and smart queue job manager. It‘s easy to install and more simple to use

Filosofy

Each client put your job on a remote queue . Each node listen this queue and grab a jone who can processes . When the job is done, two file is in the rqueue report node directory .

Installation

Installation of rqueue is very very simple and faster .

  • Put the server in yours machine server . Run rqueue_server and wait . When server starting end you simple copy the data/config.yml file in the data/ of all nodes and clients.
  • Put node_grab in evrey node of your cluster. Run node_grab and wait. No write configuration files .... nice is'n it ?
  • Put the rqueue_client in evrey client of your cluster. Modify the job yaml file and call ruby interpreter on rqueue_client . Whene the rqueue_client done your job start to work for you

Requirements

Jruby -v = all
